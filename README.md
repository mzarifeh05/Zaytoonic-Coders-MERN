# 🫒 Zaytoonic Coders

A community web app for a small group of IT students at **Al-Zaytoonah University of Jordan (ZUJ)**. The platform showcases our members, facilitates tech discussions, and provides a simple admin panel to manage the crew.

🔗 **Live Demo:** [https://zaytoonic-coders.onrender.com](https://zaytoonic-coders.onrender.com)

---

## ✨ Features

- **Member Cards** — Browse all community members with their photo, major, and bio
- **Admin Panel** — Protected by a secret key; allows adding, editing, and deleting members
- **Cloudinary Image Upload** — Drag & drop photo uploads, no manual URL entry
- **Responsive Design** — Works across desktop and mobile
- **Animated UI** — Smooth entrance animations, toast notifications, and micro-interactions
